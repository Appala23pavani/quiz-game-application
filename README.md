# 🎯 Quiz Game Application

A responsive and interactive **Quiz Game Application** built using **HTML, CSS, and JavaScript** as part of **Task 03** of my **Web Development Internship at SkillCraft Technology**.

## 📌 Project Overview

This project is a modern quiz application that allows users to test their knowledge of HTML, CSS, and JavaScript through multiple-choice questions. It provides instant feedback, tracks scores, displays a progress bar, includes a countdown timer, and stores the highest score using Local Storage.

## ✨ Features

- 📝 10 Multiple Choice Questions
- ✅ Instant Answer Validation
- ⭐ Live Score Tracking
- 🏆 High Score Saved Using Local Storage
- ⏳ 15-Second Countdown Timer
- 📊 Animated Progress Bar
- ⬅️ Previous & ➡️ Next Navigation
- 🔄 Restart Quiz
- 🌙 Dark/Light Mode Toggle
- ⌨️ Keyboard Support
- 🎉 Confetti Animation for High Scores
- 🔊 Correct & Wrong Answer Sound Effects
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/SCT_WD_03_Quiz_Game.git
```

2. Open the project folder in **Visual Studio Code**.

3. Install the **Live Server** extension (if not already installed).

4. Open the `index.html` file.

5. Right-click on `index.html` and select **Open with Live Server**.

6. The Quiz Game Application will open in your default web browser.

## 🎮 How to Play

1. Read each question carefully.
2. Select one of the four answer options.
3. Click **Next** to move to the next question.
4. Complete all questions before the timer runs out.
5. View your final score at the end.
6. Click **Restart Quiz** to play again.
